POST /api/exercise/new-user
POST /api/exercise/add

Get user's log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional
from, to = dates (yyyy-mm-dd); limit = number

Usage

POST /api/exercise/new-user will register you.

GET /api/exercise/users will give all the users in the system.

POST /api/exercise/add will add a new exercise to user's log array and returns the following:

GET /api/exercise/log?userId=someId&from=fromTime&to=untillTime will give exercise log of the user for a specified period of time.
